# AI-Based Image Data Annotation using Python

##  Project Overview

This project is an AI-powered Image Data Annotation System developed using Python and YOLOv8. It automatically detects objects in images, draws bounding boxes, assigns labels, and saves annotation details into CSV files. The project also provides data visualization and statistical analysis of detected objects.

---

##  Objective

The main objective of this project is to automate the image annotation process using the YOLOv8 object detection model. It reduces manual effort and improves the efficiency of dataset preparation for computer vision applications.

## ▶️ How to Run

1. Install the required libraries:

```bash
pip install ultralytics opencv-python pandas matplotlib pillow
```

2. Open `Image_Annotation.ipynb` in Jupyter Notebook.

3. Place all input images inside the `images` folder.

4. Run all notebook cells.

5. The annotated images will be saved in the `outputs` folder.

6. Annotation details will be saved in the `annotations` folder.
